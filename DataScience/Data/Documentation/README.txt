Dataset Title: Undergraduate Student Stress Dataset (Bangladesh)

Description:
This dataset contains 3000 anonymized responses collected from undergraduate students 
(aged 19–24) studying at public, private, and national universities in Bangladesh. 
It includes academic, lifestyle, socio-economic, and psychological attributes along with 
derived stress metrics.
